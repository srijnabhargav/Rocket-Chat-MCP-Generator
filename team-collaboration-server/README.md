# team-collaboration-server

A minimal Rocket.Chat MCP server generated by `rocket-chat-mcp-generator`.

## Tools

| Tool | Method | Path | Summary |
| --- | --- | --- | --- |
| `post-api-v1-channels_create` | `POST` | `/api/v1/channels.create` | Create Channel |
| `post-api-v1-channels_invite` | `POST` | `/api/v1/channels.invite` | Add Users to Channel |
| `post-api-v1-chat_postMessage` | `POST` | `/api/v1/chat.postMessage` | Post Message |
| `post-api-v1-login` | `POST` | `/api/v1/login` | Login with Username and Password |

## Quick Start

```bash
npm install
cp .env.example .env
npm start
```
